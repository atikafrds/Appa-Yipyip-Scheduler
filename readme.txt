--APPA SCHEDULER--
Program ini merupakan program penjadwalan dengan memanfaatkan
local search dengan tiga algoritma:
1. Hill-Climbing
2. Simulated Annealing
3. Genetic Algorithm

Cara menggunakan program:
1. Buka folder exe.
2. Jalankan executable jar file bernama "Tubes1.jar".
3. Aplikasi bernama "Appa Scheduler" akan terbuka.
4. Klik "Choose Input File!" untuk memilih file teks (.txt)
yang akan dimasukkan ke dalam program sebagai sumber informasi
jadwal.
5. Pilih salah satu algoritma yang telah tersedia.
6. Klik "Run!".
7. Hasil penjadwalan akan muncul pada tabel yang tersedia. Hasil
penjadwalan ditampilkan terkelompok berdasarkan ruangan yang ada.
8. Untuk melihat jadwal yang terbuat di ruangan lain, klik
dropdown menu di bawah tulisan "Ruangan:" dan pilih ruangan yang
diinginkan.
9. Anda dapat memindahkan jadwal secara manual dengan klik
"Change Schedule" kemudian mengikuti perintah yang diberikan.
10. Contoh hasil run program dapat dilihat di laporan pada bab
"Tampilan Program".
